Appliance Training Session

Prepare NILM training data from mains history using interval supervision or a ground-truth appliance sensor.

This page prepares an appliance model from the mains signal. Use the training server connection card first, then choose the supervision mode that matches the data you have.

  1. In "Appliance for Training", enter the Appliance Name and choose a supervision mode: ON intervals in mains if you only know when the appliance was on, or Ground-truth appliance power sensor if you have a real appliance meter.
  2. Confirm that the Training Server Connection card shows the server as ready before spending time labeling data. If not, fix the connection first and use Retry.
  3. Choose the mains history range with Quick Range or Select Date Range. You can pan and zoom the chart, and the current visible window is mirrored in the time fields beside the chart.
  4. In ON intervals in mains mode, press Draw Interval and then tap or click once for the start and once for the end. You can also edit the Start and End fields manually and press Add Interval.
  5. In Ground-truth appliance power sensor mode, select the appliance sensor and verify its power trace against the mains chart over the same range. No interval drawing is needed in this mode.
  6. Review the Selected Windows Preview when using interval supervision. Make sure the windows cover the appliance activity you want the model to learn.
  7. When everything looks correct, click Check All Data. If preparation succeeds, the session creates a ready-to-send training package for the appliance name you entered.
  8. Then click Send to Training Server to launch training and follow progress in Training Jobs. Once training finishes, the new appliance model will appear on the main configuration page.
